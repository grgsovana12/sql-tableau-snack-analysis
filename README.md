# Snack Nutrition Analysis (SQL + Tableau)

## Project Overview
This project analyzes snack nutrition data using SQL and visualizes insights using Tableau Public.

## Tools Used
- SQLite
- SQL
- Microsoft Excel
- Tableau Public

## Dataset
A small snack nutrition dataset including calories and protein information.

## Process
- Created a SQL table and inserted nutrition data
- Removed duplicate records
- Performed queries to identify highest and lowest calorie snacks
- Calculated calorie efficiency
- Built a Tableau dashboard to visualize findings
